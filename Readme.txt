== Description ==
A simple **Maintenance Mode Plugin** for WordPress. It enables a **custom maintenance page** with a **countdown timer** and admin-only access.

== Features ==
✅ Enable/Disable Maintenance Mode from Admin Panel  
✅ Custom **maintenance page** with styling  
✅ **Countdown timer** to indicate when the site will be back  
✅ Only admins can access the website while in maintenance mode  

== Screenshots ==
1. Maintenance Mode settings in the WordPress Admin panel.
2. Example of the custom maintenance page with a countdown timer.

== Installation ==
1. Upload `maintenance-mode` folder to `/wp-content/plugins/`
2. Activate the plugin in **WordPress > Plugins**.
3. Configure settings under **"Maintenance Mode"** in the Admin Menu.
